# AirSense: Air Quality Intelligence System

AirSense is a premium, full-stack air quality monitoring and predictive analytics platform. It combines real-time data from global sensors with AI-powered forecasting to help users breathe better and live healthier.

## 🚀 Core Features
- **Real-Time Monitoring**: Live AQI and pollutant concentrations for any city worldwide.
- **AI-Powered Forecasting**: 7-day predictive analytics using a Random Forest ML ensemble.
- **Cinematic Experience**: High-fidelity GSAP animations and WebGL atmospheric backgrounds.
- **Secure Authentication**: JWT-protected user accounts with encrypted data persistence.

## 🛠️ Technology Stack
- **Frontend**: React 19, GSAP, OGL (WebGL), Vite.
- **Backend**: Flask (Python), Scikit-Learn, Matplotlib.
- **Database**: MongoDB Atlas.

## 🏁 Getting Started
1. Run `Launch_AQI_Monitor.bat` in the root directory.
2. The system will automatically initialize the Backend and Frontend servers.
3. Open `http://localhost:3000` in your browser.
