# AQI Monitor — Cinematic Frontend (React)

Welcome to the world-class frontend for the **AQI Monitor System**. This application is built with a focus on immersive UX, cinematic performance, and data-driven insights.

## 🚀 Key Features

- **Cinematic Scroll Architecture**: A unified GSAP-driven scroll experience that transforms the landing page into a narrative journey.
- **Atmospheric Backgrounds**: A custom WebGL "Plasma" effect that shifts colors (Emerald → Amber → Deep Blue) as you navigate between sections.
- **Tactile Interactivity**: 
    - **Magnetic Buttons**: CTAs that dynamically react to mouse proximity.
    - **Interactive Glare**: Feature cards with light-tracking "glass" surfaces.
- **Live Data Integration**: Real-time AQI monitoring powered by the OpenWeather API.
- **AI Forecasting**: Visualizing 7-day predictions generated by the Python backend.

## 🛠️ Technology Stack

- **Framework**: React 19 + Vite
- **Animations**: GSAP (GreenSock) + ScrollTrigger
- **Visuals**: OGL (WebGL framework for the background)
- **Icons**: FontAwesome 6
- **Typography**: Playfair Display (Serif) & Nunito Sans (Interface)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Navigate to the `react-app` directory:
   ```bash
   cd pollution-app/react-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure
- `src/pages/Home`: Contains the primary landing page sections (Hero, Features, Process, Control).
- `src/components`: Reusable UI elements (Navbar, Footer, Plasma, ScrollProgress).
- `src/styles`: Modular CSS focusing on "Apple-style" premium aesthetics.
